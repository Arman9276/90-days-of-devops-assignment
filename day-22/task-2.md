Task 2 — Create Your Git Project
📁 1. Create Folder devops-git-practice

Go to the location where you want your project (example: Documents):

cd ~/Documents
mkdir devops-git-practice
cd devops-git-practice

Check folder:

pwd
ls
🔧 2. Initialize it as a Git Repository

Run:

git init

✅ Output will look like:

Initialized empty Git repository in .../devops-git-practice/.git/

👉 This creates a hidden .git directory — this is the brain of Git.

🔎 3. Check Git Status (Important)
git status

You will see something like:

On branch master (or main)

No commits yet
nothing to commit (create/copy files and use "git add" to track)
🧠 Understand What Git is Saying

On branch main/master → Your current branch

No commits yet → Repo is empty

nothing to commit → No tracked files yet

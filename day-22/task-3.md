# Git Commands Reference

This file contains the Git commands used so far during DevOps Git practice.

---

## ⚙️ Setup & Config

### `git --version`

**What it does:** Shows the installed Git version on your system.

**Example:**

```bash
git --version
```

### `git config --global user.name`

**What it does:** Sets your global Git username for commits.

**Example:**

```bash
git config --global user.name "Arman Narsinh"
```

### `git config --global user.email`

**What it does:** Sets your global Git email address.

**Example:**

```bash
git config --global user.email "your-email@example.com"
```

### `git config --list`

**What it does:** Displays all Git configuration settings.

**Example:**

```bash
git config --list
```

---

## 🔧 Basic Workflow

### `git init`

**What it does:** Initializes a new Git repository in the current directory.

**Example:**

```bash
git init
```

### `git status`

**What it does:** Shows the current state of the working directory and staging area.

**Example:**

```bash
git status
```

---

## 👀 Viewing Changes

### `git status`

**What it does:** Displays tracked/untracked files and pending changes.

**Example:**

```bash
git status
```
